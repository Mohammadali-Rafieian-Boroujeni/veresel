# Translations

Drop `veresel-{locale}.po`/`.mo` files here (e.g. `veresel-fa_IR.po`).

If this plugin is hosted on WordPress.org, translations are normally handled
automatically via https://translate.wordpress.org/ and this folder is not
needed for that - it only matters for self-hosted/zip installs using
`load_plugin_textdomain()` (see veresel.php).
